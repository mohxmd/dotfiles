# Neovim (LazyVim)

## Start

```bash
nvim
```

## File Tree (Right Sidebar)

- Toggle sidebar: `<leader>e`
- Reveal current file: `<leader>fe`
- If needed: `:Neotree toggle right`

## Basic

- Save: `:w`
- Quit: `:q`
- Save + quit: `:wq`

## Plugins / LSP

- Sync plugins: `:Lazy sync`
- Update plugins: `:Lazy update`
- Check health: `:checkhealth`
- Manage LSP tools: `:Mason`
